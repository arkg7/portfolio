# portfolio
This repository is for my personal Portfolio Website, including a homepage, projects page, and an 'about me' page. 

This project features heavy use of HTML and CSS to create a stylized personal experience, combined with custom assests and assets found online from providers such as FontAwesome and Google Fonts.

This site demonstrates a understanding of user-interactivity and mixes it with personal expression to showcase my abilities to create, update, and maintain web projects while displaying my personality.
